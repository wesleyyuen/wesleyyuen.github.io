VTuber Studio

An Open project in Unreal Engine that provides a template for creating a virtual studio for streaming, leveraging different OffWorld Live and 3rd-party plugins to provide:

  • Customizable cameras (Cinecam, 360 etc.)
  • Media input/output options (via NDI, Spout etc.)
  • Motion capture for metahumans via webcam

The project is open for anyone to share useful custom blueprints in their workflow with the rest of the OffWorld Live community
